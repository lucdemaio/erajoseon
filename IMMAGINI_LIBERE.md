## 💾 Come Inserire Nuove Immagini nel Sito

Per aggiungere una nuova immagine nel file HTML:

```html
<img src="URL_IMMAGINE" alt="Descrizione" class="page-image">
<p class="image-credit">Titolo immagine - Foto Unsplash</p>
```

### Esempio completo:
```html
<img src="https://images.unsplash.com/photo-1234567890123-abcdef123456?w=400&q=80" 
     alt="Palazzo reale coreano" 
     class="page-image">
```
